This is a console-based messaging application called QuickChat. After login, users can send messages with proper validation. The system includes Message ID generation, recipient number validation, message hash creation, and options to send, disregard, or store messages. It also tracks the total number of messages sent.

Files Included:
•  Login2.java
•  LoginSystem2.java
•  Message.java
•  QiuckChat.java (Main class)

The application follows the assignment requirements for user login and message handling.
